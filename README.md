# crypto-toolkit-python
README

## What this shows
- Practical Python scripting
- Secure handling basics (no hardcoded secrets, clear usage)
- Understanding of cryptographic primitives and trade-offs

## Features
- Hashing: SHA-256 (with optional salt)
- Symmetric encryption: Fernet (AES under the hood)
- Asymmetric encryption: RSA keypair + encrypt/decrypt
